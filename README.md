# News Hub

A simple news website powered by [Jekyll](https://jekyllrb.com).

## Local development

1. Install dependencies:
   ```bash
   bundle install
   ```
2. Build the site:
   ```bash
   bundle exec jekyll build
   ```
3. Serve locally:
   ```bash
   bundle exec jekyll serve
   ```

Generated pages will be written to the `_site` directory.
